# PurdueAirShare
Final Project for Tech 120: Design in Thinking Technology 

Developed a full-stack prototype that enables Purdue students to book cost-shared aircraft or ground rides to major cities (e.g., Chicago, Indianapolis).
Built with FastAPI (Python) for backend APIs and React Native + Expo for a modern, cross-platform mobile UI.

Key highlights:


Designed a real-time flight search and booking interface with airport-code routing (KLAF → KORD).

Implemented RESTful API endpoints using FastAPI with dynamic query filtering and simulated flight data.

Created a responsive dark-themed Purdue-branded UI using React Native, Tailwind-style layouts, and Expo components.

Integrated CORS, JSON validation (Pydantic), and modular backend architecture ready for cloud deployment.

Showcases cross-domain engineering — combining aviation data modeling, modern UX, and secure backend design.

Tech Stack:
FastAPI · Python · React Native · Expo · Node.js · JavaScript (ES6) · REST APIs · UI/UX Design
